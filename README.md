# PUBG Map Overlay (배그 맵 오버레이)

배틀그라운드(PUBG) 게임 플레이 중 지도 위에 정확한 위치 표시를 도와주는 오버레이 프로그램입니다.
QHD 해상도를 기준으로 정밀 보정되었으며, 단축키를 통해 게임 화면 위에 원(반경 1200m)을 표시합니다.

## ✨ 주요 기능
- **오버레이 토글**: 게임 중 단축키로 원 그리기/숨기기
- **간편한 설정**: 시스템 트레이(시계 옆 아이콘) 또는 단축키로 설정 창 진입
- **클릭 투루(Click-Through)**: 오버레이가 켜져 있어도 마우스 클릭이 게임으로 전달됨
- **정밀 보정**: QHD(1440p) 해상도에서 격자(1km) 기준 1.2배(1200m) 정밀도 확보
- **시스템 독립적 설정**: 사용자 설정(`config.ini`)은 AppData 폴더에 안전하게 보관됨

## 🚀 사용 방법

### 1. 실행
`PUBG_Map_Overlay.exe`를 실행하면 시스템 트레이에 아이콘이 생깁니다.

### 2. 기본 단축키
| 기능 | 기본 단축키 | 설명 |
|---|---|---|
| **보이기/숨기기** | `F8` | 원을 화면에 띄우거나 숨깁니다. |
| **설정 열기** | `F12` | 설정 창을 엽니다. |
| **프로그램 종료** | `Alt + Q` | 프로그램을 즉시 종료합니다. |

*(단축키는 설정 창에서 원하는 조합으로 변경 가능합니다. 변경 시 `config.ini`에 저장됩니다.)*

### 3. 설정 (Calibration)
원의 위치나 크기를 미세하게 조정하고 싶다면 **설정(F12)** 창을 이용하세요.
- **Scale Factor**: 전체적인 오버레이 크기 배율 (기본 1.0)
- **Offset X/Y**: 원의 중심 위치 이동

```text
.
├── assets/             # 실행에 필요한 리소스 (아이콘, 오버레이 이미지)
├── tools/              # 개발 도구 (오버레이 재생성 스크립트 등)
├── main.py             # 메인 애플리케이션 코드
├── build.py            # PyInstaller 빌드 스크립트
└── README.md
```

## 🛠 개발 및 빌드 (Windows)
직접 수정하거나 빌드하려면 다음 준비물이 필요합니다:

1. **Python 3.10+** (및 **uv** 패키지 매니저 권장)
2. **의존성 설치**:
   ```powershell
   uv sync
   ```
3. **오버레이 이미지 재생성** (필요 시):
   `/tools/generate_overlay.py`를 실행하면 `/assets/overlay_circle.png`가 업데이트됩니다.
4. **빌드**:
   ```powershell
   uv run build.py
   ```
   실행 파일은 `dist/` 폴더 내에 생성됩니다.

## ❓ 문제 해결 (Troubleshooting)
- **오버레이가 안 보여요!**
    - 게임 설정을 **"전체 화면(창)"** 또는 **"창 모드"**로 변경하세요.
- **설정이 초기화돼요!**
    - 설정 파일 위치: `%LOCALAPPDATA%\PUBG_Map_Overlay\config.ini`
    - 앱 버전이 업데이트되면 호환성을 위해 설정이 초기화될 수 있습니다.
